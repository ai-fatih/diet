# Связь: эндометриоз ↔ вестибулярные симптомы

## Доказательства

| Связь | PMID | Тип |
|-------|------|-----|
| Dizziness у 44.8% пациенток с эндометриозом | 39959963 | case-control |
| Эндометриоз → нейровоспаление через блуждающий нерв | 42193132, 42222530 | review |
| Shared genetics: эндометриоз + мигрень = общие воспалительные пути | 42222530 | review |
| Endometrial polyps → риск вертиго (aOR=1.04-1.05) | 41504513 | cross-sectional |

## Гипотетические механизмы

1. **Systemic inflammation**: эндометриоз → ↑ цитокины (IL-6, TNF-α) → воспаление вестибулярного нерва
2. **Neuroinflammation**: эндометриоз → активация микроглии → центральная сенситизация → dizziness
3. **Vagus nerve**: эндометриоидные очаги → афферентная стимуляция n.vagus → нарушение вестибулярной компенсации
4. **SIBO → systemic**: кишечник → эндотоксины → TLR4 → NF-κB → воспаление → ВН
