# Vestibular Rehabilitation Dashboard

Персонализированный медицинский дашборд и база знаний по вестибулярной реабилитации.

**Сайт:** https://vestibular-rehabilitation.vercel.app

## О проекте

Открытая публичная документация клинического случая вестибулярного неврита (ICD-11: AB31.0). Включает:

- **Дашборд** — персонализированные медицинские данные, анализы, диагностика
- **VRT-протокол** — упражнения для вестибулярной реабилитации (VOR, Cawthorne-Cooksey)
- **Научная база** — обзор вестибулярной нейроиммунологии, патогенеза, регуляторных механизмов
- **Инструменты** — конфигуратор таймлайна фаз реабилитации, карта пациента

Все данные представлены в открытом доступе для образовательных и исследовательских целей.
