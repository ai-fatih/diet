# Эндометриоз

## Факты из литературы

- Хроническое воспалительное, эстроген-зависимое заболевание
- 5-10% женщин репродуктивного возраста
- Связан с системным воспалением — не только локальный процесс
- 91.9% пациенток с эндометриозом имеют SIBO/IMO (PMID: 39959963)
- Dizziness/головокружение — у 44.8% пациенток с эндометриозом (PMID: 39959963)
- Имеет общие генетические воспалительные пути с мигренью (PMID: 42222530)
