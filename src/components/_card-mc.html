★ Приоритет ◇ Стабилизация тучных клеток

🥗 Низкогистаминовая диета SIGHI ×3–4 нед

✓ Исключить: ферментированное, выдержанные сыры, алкоголь, консервы, остатки >24ч
✓ Основы: свежие овощи, зелень, куркума, имбирь, лук
✓ Свежее мясо/рыба — приготовить в день покупки

💊 Нутрицевтики для стабилизации MC

✓ Кверцетин 2000–4000 мг/д (флавонол, ингибирует дегрануляцию)
✓ Лютеолин 100–200 мг/д (синергист кверцетина, ингибирует CRHR-1)
✓ PEA (пальмитоилэтаноламид) 400–800 мг/д (агонист PPAR-α, ↓ активацию MC, синергист кверцетина)
✓ Витамин C 1–3 г/д (антиоксидант, ↓ гистамина)
✓ Витамин D 2000–5000 МЕ/д (иммуномодуляция MC)
✓ NAC 500–1000 мг/д (предшественник глутатиона, защита BLB)
✓ Куркумин 1–4 г/д (↓TNF-α, ↓ активации MC, пиперин для усвоения)

🧪 Мишени воздействия

• MRGPRX2 — рецептор SP на MC → блокируется кверцетином
• CRHR-1 — рецептор CRH на MC → блокируется лютеолином
• ↓ гистамина → ↓ проницаемости BLB → ↓ отёка нерва
