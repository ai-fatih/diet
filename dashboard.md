## 4. Дифференциальная диагностика (гипотезы)

| Гипотеза | Вероятность | Обоснование |
|---|---|---|
| **Гемангиомы позвонков С4-С5** | 🔴 Наиболее вероятно | Доброкачественные сосудистые образования. Частая находка на КТ (после 30–40 лет). Не влияют на анализы крови. |
| **Цервикогенное головокружение** | 🔴 Высокая | Связь с болью в шее, положительная динамика от массажа, улучшение в положении лёжа. |

| **Локальные фиброзные/жировые включения** | 🟡 Умеренная | Особенность строения костной ткани, не патология. На КТ выглядят как зоны сниженной плотности. |
| **Метастатическое поражение** | 🟢 Крайне маловероятно | Нормальные СРБ, СОЭ, ферритин, гемоглобин. Отсутствие первичной опухоли по данным маммографии и КТ головы. |

## 5. Рекомендации

### Приоритетные действия

1. **МРТ шейного отдела позвоночника** — определяет природу очагов в С4-С5 (гемангиома/фиброз/жир/другое). **Золотой стандарт.** Без него упражнения для шеи противопоказаны.
2. **Приём невролога** (запланирован) — с результатами анализов крови и КТ. Запросить пробу на цервикогенное головокружение.
3. **УЗИ сосудов шеи (БЦА)** — по назначению невролога, для оценки кровотока при поворотах головы.

### Образ жизни

| Что делать | Что не делать |
|---|---|
| Увеличить питьевой режим (вода) | Не делать активных упражнений для шеи до МРТ |
| Ортопедическая подушка с валиком под шею | Не спать на животе |
| Изометрическая гимнастика («двойной подбородок» — задвигать подбородок назад без наклона) | Не делать резких поворотов/наклонов головы |
| Диафрагмальное дыхание (животом) для снижения тревоги | Не игнорировать боль в шее |
| Приём магния (глицинат/цитрат, 300–400 мг/сут) — продолжать | — |
| Массаж шеи (мягкий, без усилия) — продолжать | — |
| Ведение дневника головокружения (когда/что провоцирует/что помогает) | — |
